#### Install script on Termux
```bash
$ pkg update && pkg upgrade
$ pkg install python2
$ pkg install git
$ git clone https://github.com/MRVIVEK-CODER/XDF
$ pip2 install requests bs4
$ pip2 install futures
```
#### Run script
```bash
$ cd XDF
$ python2 XDF.py
```
